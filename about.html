Hey! I'm Trying to Code.



Why I made this...

Code is intuitive. It is logical.

I've been coding for 5 years, and along the way it was a struggle.

I thought there might be a better way to start people into actual code.

I've been testing this site for a little while and I think it's working. I want this site to be a place where people can go to intuitively understand what it is like to code, try some learning projects, get hooked into the fun part of code, and eventualy dive deeper into code on other sites. When I started, I struggled to understand what code really was, what it was for, and what I had to do in order to learn it.

I want a beginner to be able to go here, and feel like they've always known code.

Here is my plan:

- keep the base features free forever

- allow users to remain anonomous (no email)

- listen to user feedback

- focus on specific audiences

- be simplistic and fun

Who is this for?

At the moment I have 2 audiences in mind:



  • Teachers - high school teachers helping young students first start code.


  • Students - total beginners, who want to understand what coding is like quickly.
